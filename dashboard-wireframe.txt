INTERACTION NOTES:
==================
- Drag handles (≡) for reordering blocks in Shots section
- Up/Down arrows for fine control of block order
- × button for deleting blocks
- File upload areas support drag & drop
- Parameters section supports various input types (text, dropdown, slider, toggle)
- Toolset provides expandable tool selection
- All text areas are resizable
- Video blocks show preview/playback controls
